# First Steps

### git clone (`https://github.com/gururajm1/youtube-playlister.git`)

### `cd playlister`

Navigate to the Client Directory

### `npm install`

Install Node Packages

### `npm start`

Start the client 


### `cd playlister-server`

Navigate to the Server Directory

### `npm install`

Install Node Packages

### `npm start`

Start the server js
